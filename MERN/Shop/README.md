
Concurrent:
```
npm i concurrently
```
Add scripts to package.json:
```
"scripts": {
    "start": "node index.js",
    "server": "nodemon index.js",
    "client": "npm run start --prefix client",
    "devClient": "concurrently -k \"npm run server\" \"npm run client\""
  },
```
And then run:
```
npm run devClient
```

## GIT : Switch to existing branch or to new branch
```
$ git checkout <existing_branch>

$ git checkout -b <new_branch>
```

## GIT : Show current branch
```
$ git branch --show-current
```

## GIT : Delete remote branch
```
git push -d <remote_name> <branchname>
```

## GIT : Delete local branch
```
git branch -d <branch_name>
git branch -D <branch_name>
```
The -d option is an alias for --delete, which only deletes the branch if it has already been fully merged in its upstream branch.
The -D option is an alias for --delete --force, which deletes the branch "irrespective of its merged status."
